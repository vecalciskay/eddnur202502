Hacer un programa en Java que muestre una
ventana con la lista Estudiante/Persona que 
se manejará.

Al lado izquierdo o derecho de cada item en esa
lista, debe haber un boton para editar el registro 
y otro para eliminar el registro.

Cuando se hace clic en editar aparece un popup 
y se puede colocar los datos del Estudiant/Persona.
Esto para el insert / update.

Adicionalmente, se debe programar el
eliminar de manera similar.
